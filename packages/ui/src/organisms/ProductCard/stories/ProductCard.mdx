## Components

The `ProductCard` uses the [Compound Component](https://kentcdodds.com/blog/compound-components-with-react-hooks) pattern, its components are:

- `ProductCard`: the wrapper component;
- `ProductCardLink`: the component that handles call-to-action link;
- `ProductCardImage`: the wrapper component for the product's image;
- `ProductCardInfo`: the wrapper component that can receive a `ProductCardHeader`, `ProductCardPrice` or `ProductCardTags`;
- `ProductCardHeader`: the wrapper component for the product's title and description;
- `ProductCardPrice`: the wrapper component for the product's price;
- `ProductCardTags`: the wrapper component for the product's tags, i.e. discount tags.
